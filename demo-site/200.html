How it works

Scroll Morph

Lines near the center of the viewport render large and bright. Lines at the edges shrink and fade. The result is a natural reading focus that follows your scroll position.

Pinch to Zoom

Two-finger pinch gestures on mobile scale the text range in real time — no page zoom, no layout shift. Just bigger or smaller text, instantly.

Accurate Line Breaking

Text is measured and wrapped using @chenglou/pretext for segment-aware, pixel-perfect line breaking on canvas.

Momentum Scrolling

Touch scrolling with configurable friction and edge bounce. Feels native. Works everywhere.

Install

$ npm install pinch-type @chenglou/pretext
import { createPinchType } from 'pinch-type'; const pt = createPinchType(document.getElementById('reader')); pt.setText('Your long article text here…'); // Pinch-to-zoom works automatically on mobile. // Call pt.destroy() to clean up.

Options

createPinchType(element, { // Font size at viewport center (default: 26) initialFontSize: 26, // Font size at viewport edges (default: 11) edgeFontSize: 11, // Min/max zoom bounds minFontSize: 8, maxFontSize: 60, // Scroll momentum friction (0–1) friction: 0.95, // Morph gradient radius in px morphRadius: 300, // Zoom callback onZoom: (center, edge) => console.log({ center, edge }), });
